# Assets

This folder contains the site's assets.

## Placeholder Images

The site currently references `placeholder.svg` for all images.

**To use the site fully:**

1. Replace `placeholder.svg` with real images, or
2. Use a placeholder service such as [placeholder.com](https://placeholder.com/) or [via.placeholder.com](https://via.placeholder.com/)
3. Or create your own images with the appropriate dimensions:
   - Blog images: 16:9 (e.g., 1200x675px)
   - Album images: 1:1 (e.g., 800x800px)
   - Biography photo: 1:1 (e.g., 600x600px)
   - Tarot cards: 3:5 (e.g., 300x500px)

## Temporary placeholder example

If you need something quick for testing, you can create a simple `placeholder.svg` with any image editor at 1200x675px (16:9).
